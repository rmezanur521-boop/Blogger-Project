# 📝 Blogger Project

A full-featured blogging web application built with ASP.NET MVC and SQL Server.  
It allows users to create, edit, and manage blog posts with authentication, CRUD operations, and role-based access control.

---

## 🚀 Features

- 🔐 User Authentication (Login / Register)
- 📝 Create Blog Posts
- ✏️ Edit & Update Posts
- ❌ Delete Posts
- 📋 View All Blogs
- 👤 Role-based Access (Admin / User)
- 🛠️ Full CRUD Operations

---

## 🛠️ Technologies Used

- ASP.NET MVC
- C#
- SQL Server
- Entity Framework
- HTML, CSS, Bootstrap
- JavaScript
